1) pwd
comando = pwd
Que hace: muestra el directorio actual

2) ls 
comando = ls
Muestra la lista de archivo

3) cd
comando = cd tp1-mi-sitio
se uso para entrar en una carpeta del proyecto

4) mkdir
comando = mkdir images
lo use para crear una carpeta de imagenes

5) touch
comando = touch (touch index.html)
este comando crea un archivo vacio pero lo use para crear un archivo .html

6) cat
comando = cat 
muestra un archivo en la terminal
se uso para revisar el contenido del archivo (cat index.html)

7) which
comando = which git
muestra la ubicacion en donde esta instalado un programa
lo use para git   (which git)

8) code
comando = code
abre el visual studio code al poner ese comando en la terminal 

9) clear
limpia la pantalla de la terminal

10) git --version
se uso para mostrar la version del programa git y para verificar que este instalado el programa 
correctamente
